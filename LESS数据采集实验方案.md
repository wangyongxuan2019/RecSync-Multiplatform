# LESS数据采集实验方案

## Landing Error Scoring System (LESS) 数据采集标准化流程

---

## 1. 研究概述

### 1.1 研究目的

**核心目标**：构建高精度的LESS自动评估系统，实现ACL损伤风险的智能化、客观化筛查。

传统LESS评估依赖专业人员主观判断，存在以下局限：
- 需要经过专业培训的评估人员
- 评估者间一致性难以保证
- 耗时耗力，难以大规模推广
- 无法提供实时反馈

本研究旨在通过计算机视觉与深度学习技术，开发一套**端到端的自动化LESS评估系统**，使其能够：
- 从双视角视频中自动提取人体姿态
- 精准检测关键帧（初始触地IC、最大屈膝MKF）
- 自动计算LESS 17项指标评分
- 输出ACL损伤风险等级及改进建议

为实现上述目标，本阶段需完成高质量数据集的采集与标注工作，具体包括：
1. 采集标准化的双视角同步DVJ测试视频
2. 建立专家标注的关键帧与LESS评分金标准
3. 构建用于模型训练与验证的基准数据集

### 1.2 测试动作

采用**落地跳跃测试（Drop Vertical Jump, DVJ）**作为标准测试动作，该测试是LESS评估的标准范式，具有以下特点：
- 标准化程度高，便于重复和比较
- 能够有效激发下肢生物力学特征
- 与ACL损伤风险高度相关
- 国际运动医学领域广泛认可

### 1.3 数据规模规划

| 数据集类型 | 受试者数量 | 每人试次 | 总视频数 | 用途 |
|-----------|-----------|---------|---------|------|
| 训练集 | 80人 | 5次 | 800组(1600个视频) | 模型训练 |
| 验证集 | 20人 | 5次 | 200组(400个视频) | 超参数调优 |
| 测试集 | 20人 | 5次 | 200组(400个视频) | 最终评估 |
| **合计** | **120人** | **5次** | **1200组(2400个视频)** | - |

> 注：每组包含正面和侧面两个同步视频

## 2. 受试者

### 2.1 目标人群

| 项目 | 人数 | 男性 | 女性 | 运动水平 |
|------|------|------|------|---------|
| 篮球 | 25 | 12 | 13 | 专业 |
| 足球 | 25 | 10 | 15 | 专业 |
| 排球 | 20 | 8 | 12 | 专业 |
| 田径（跳跃类） | 15 | 7 | 8 | 专业 |
| 羽毛球 | 15 | 6 | 9 | 专业 |
| 普通运动人群 | 20 | 7 | 13 | 非专业 |
| **合计** | **120** | **50（42%）** | **70（58%）** | - |

**选择依据**：
- **项目选择**：LESS评估的核心是落地动作的生物力学质量，ACL损伤最常发生在涉及跳跃落地、急停变向的运动中。篮球、足球、排球是ACL损伤研究中证据最充分的三类运动；田径跳跃类提供高强度落地样本；羽毛球拓展非对抗项目的动作多样性；普通运动人群确保系统在校园体测、社区筛查等场景的泛化能力。
- **女性比例（58%）**：女性ACL损伤风险是男性的2-8倍，是ACL预防研究的核心关注群体。
- **运动水平**：专业运动员指注册运动员或高校校队成员（系统训练≥2年）；非专业运动员指有规律运动习惯的普通人群。

**样本量**：120人 × 5次/人 = 1200组样本（2400个视频），考虑10-15%的数据剔除，有效样本预计≥1000组。

### 2.2 纳入标准

1. **年龄**：18-30岁
2. **性别**：男女均可
3. **运动背景**：
   - 来自上述目标运动项目
   - 能够独立完成跳跃落地动作
   - 近6个月内保持规律训练或运动习惯
4. **健康状况**：
   - 无下肢骨骼、关节、肌肉急性损伤（近6个月内）
   - 无神经系统疾病
   - 无影响平衡的前庭系统疾病
5. **知情同意**：
   - 自愿参与并签署知情同意书

### 2.3 排除标准

1. **损伤史**：
   - 既往ACL损伤或重建手术史
   - 近12个月内下肢骨折、韧带撕裂、半月板损伤
   - 近6个月内下肢肌肉拉伤或关节扭伤
   - 测试当日存在下肢疼痛或不适
2. **疾病史**：
   - 心血管系统疾病
   - 神经肌肉系统疾病
   - 平衡功能障碍
3. **其他**：
   - 孕期或哺乳期女性
   - BMI > 30 kg/m²
   - 无法理解或遵循测试指令
   - 测试当天身体不适（自我报告）

## 3. 实验场地与设备

### 3.1 场地要求

#### 3.1.1 空间要求
- **最小面积**：5m × 5m
- **净空高度**：≥3m
- **地面**：平整、防滑的运动地板或体操垫

#### 3.1.2 环境要求
- **光照**：均匀明亮，避免强烈阴影
  - 推荐照度：500-1000 lux
  - 避免逆光和侧向强光
  - 优先选择漫射光源
- **背景**：
  - 纯色背景（推荐浅灰、浅蓝）
  - 避免复杂图案和反光材质
  - 与受试者服装形成对比
- **温度**：20-25°C（适宜运动）
- **噪音**：安静环境，便于指令传达

### 3.3 相机设置

#### 3.3.1 RecSync系统配置

**Leader端设置**：
- 视频分辨率：1920×1080（推荐）或1280×720
- 帧率：60fps（推荐）或30fps
- 受试者ID格式：s01, s02, s03...
- 动作ID：m01（DVJ测试）
- 回合自动递增：e1, e2, e3...

**Client端命名**：
- 正面相机：`front`
- 侧面相机：`side`

**生成文件示例**：
```
front_s01_m01_20240115143022_e1.mp4  (正面视角)
side_s01_m01_20240115143022_e1.mp4   (侧面视角)
```

#### 3.3.2 相机位置参数

| 参数 | 正面相机 (front) | 侧面相机 (side) |
|------|-----------------|-----------------|
| 距离 | 3-4m | 3-4m |
| 高度 | 0.8-1.0m（膝关节高度） | 0.8-1.0m |
| 角度 | 垂直于额状面 | 垂直于矢状面 |
| 视野 | 完整覆盖跳箱至落地区 | 完整覆盖跳箱至落地区 |

#### 3.3.3 画面要求

- [ ] 受试者全身在画面内（头顶至脚底）
- [ ] 跳箱和落地区完整可见
- [ ] 画面稳定，无抖动
- [ ] 曝光适中，细节清晰
- [ ] 无运动模糊（快门速度足够）
- [ ] 两路视频时间同步（RecSync自动保证）

## 4. 实验流程

### 4.1 实验前准备（测试日前）

#### 4.1.1 受试者筛选与预约
1. 发布招募信息，收集报名
2. 电话/问卷初筛，确认符合纳入标准
3. 预约测试时间，每人预留30分钟
4. 发送测试须知（着装、注意事项）

#### 4.1.2 测试须知（发送给受试者）

```
【LESS测试须知】

测试时间：____年__月__日 __:__
测试地点：________________

请您注意以下事项：

1. 着装要求：
   - 穿着贴身的运动短裤/紧身裤（便于观察膝关节）
   - 穿着对比色上衣（避免与背景同色）
   - 穿着平底运动鞋（非气垫、非高帮）
   - 女性请穿运动内衣

2. 测试前准备：
   - 测试前2小时避免剧烈运动
   - 测试前30分钟避免大量进食
   - 保证充足睡眠

3. 测试当天：
   - 请提前10分钟到达
   - 如身体不适请提前告知

联系人：________ 电话：____________
```

#### 4.1.3 场地与设备准备

**测试日前一天**：
- [ ] 检查场地可用性
- [ ] 检查所有设备功能正常
- [ ] 充电/备用电池
- [ ] 准备记录表格
- [ ] 确认受试者名单

**测试日当天（提前1小时）**：
- [ ] 布置场地，设置跳箱位置
- [ ] 架设相机，调整角度
- [ ] 启动RecSync系统，测试同步
- [ ] 试录一段视频，检查画质
- [ ] 准备急救设备

### 4.2 实验当天流程

#### 4.2.1 流程总览

```
受试者到达 → 知情同意 → 基本信息采集 → 着装检查 →
热身活动 → 动作讲解 → 练习试跳 → 正式测试 →
数据确认 → 测试结束
```

#### 4.2.2 详细流程

**第一阶段：签到与准备（约10分钟）**

| 步骤 | 时间 | 内容 | 负责人 |
|------|------|------|--------|
| 1.1 | 2min | 签到，核对身份 | 接待员 |
| 1.2 | 5min | 介绍研究目的，签署知情同意书 | 主试 |
| 1.3 | 3min | 采集基本信息（年龄、身高、体重、运动史） | 主试 |

**基本信息采集表**：

| 编号 | 项目 | 记录 |
|------|------|------|
| 1 | 受试者编号 | s____ |
| 2 | 性别 | □男 □女 |
| 3 | 年龄 | ____岁 |
| 4 | 身高 | ____cm |
| 5 | 体重 | ____kg |
| 6 | 优势腿 | □左 □右 |
| 7 | 运动类型 | ________ |
| 8 | 运动频率 | ____次/周 |
| 9 | 损伤史 | □无 □有：________ |

**第二阶段：热身与准备（约8分钟）**

| 步骤 | 时间 | 内容 |
|------|------|------|
| 2.1 | 3min | 一般热身：慢跑、开合跳 |
| 2.2 | 3min | 动态拉伸：腿部摆动、弓步走 |
| 2.3 | 2min | 专项热身：原地小跳、轻度跳跃 |

**第三阶段：动作讲解与练习（约5分钟）**

**标准动作讲解**：
1. 站在跳箱上，双脚与肩同宽
2. 双手叉腰或自然下垂
3. 向前跳下（不是跳远，是跳下）
4. 双脚同时落地
5. 落地后立即进行最大努力垂直跳跃
6. 垂直跳跃后自然落地

**演示与强调**：
- 主试现场演示标准动作
- 强调落地后"立即"反跳（不要停顿）
- 强调尽全力向上跳
- 不要刻意控制落地姿态（自然即可）

**练习试跳**：
- 允许1-2次练习跳跃
- 确认受试者理解动作要领
- 解答受试者疑问

**第四阶段：正式测试（约7分钟）**

| 试次 | 操作流程 |
|------|---------|
| 准备 | 受试者站上跳箱，调整站位 |
| 开始 | 主试口令"准备...跳！"，同时按下RecSync录制按钮 |
| 执行 | 受试者完成跳下-落地-反跳-落地 |
| 结束 | 动作完成后，主试按下停止录制 |
| 间隔 | 休息30秒，准备下一次 |

**录制操作流程（RecSync）**：

```
1. Leader端：确认受试者编号（如s01）
2. Leader端：确认动作编号（m01）
3. Leader端：确认两个Client已连接且同步
4. 口令"准备...跳！"
5. Leader端：点击"开始录制"按钮
6. 受试者完成动作（约3-5秒）
7. Leader端：点击"停止录制"按钮
8. 确认视频已自动上传到Leader端
9. 快速回放检查视频质量
10. 回合号自动递增（e1→e2→e3...）
```

**每人完成5次有效试跳**：
- 如出现以下情况，该次不计，重新测试：
  - 单脚先落地
  - 落地后未进行反跳
  - 录制设备故障
  - 受试者动作明显失误
- 使用RecSync的"重测"功能记录（生成_retake后缀）

**第五阶段：结束（约2分钟）**

1. 感谢受试者参与
2. 询问测试过程中是否有不适
3. 发放补偿（如适用）
4. 告知后续安排

### 4.3 RecSync操作详细指南

#### 4.3.1 系统启动

```
1. 启动Leader端电脑，运行RecSync-Leader.exe
2. 启动两台Client端电脑，运行RecSync-Client.exe
3. 确认Client自动发现并连接Leader
4. 在Leader端确认：
   - front设备：已连接，已同步
   - side设备：已连接，已同步
```

#### 4.3.2 参数设置

```
Leader端 → 设置：
- 分辨率：1920x1080
- 帧率：60fps
- 存储路径：D:\LESS_Data\

实验数据设置：
- 受试者ID：s01（每位受试者递增）
- 动作ID：m01（固定，DVJ测试）
- 回合：自动从e1开始递增
```

#### 4.3.3 录制流程

```
每位受试者测试流程：

1. 设置受试者ID（如：s01）
2. 第1次试跳：
   - 点击"开始录制" → 受试者执行 → 点击"停止录制"
   - 文件自动命名：front_s01_m01_时间戳_e1.mp4
   -               side_s01_m01_时间戳_e1.mp4
3. 第2-5次试跳：
   - 回合号自动递增（e2, e3, e4, e5）
4. 如需重测：
   - 点击"重测"按钮
   - 文件命名：..._e3_retake1.mp4
5. 完成后，更新受试者ID为下一位（s02）
```

## 5. 数据质量控制

### 5.1 实时质量检查

**每次录制后立即检查**：

| 检查项 | 合格标准 | 不合格处理 |
|--------|---------|-----------|
| 画面完整性 | 全身完整可见 | 重新调整相机，重测 |
| 画面清晰度 | 无模糊、无过曝 | 调整相机设置，重测 |
| 同步状态 | RecSync显示"已同步" | 重新同步，重测 |
| 动作完整性 | 跳下+落地+反跳完整 | 重测 |
| 动作规范性 | 双脚同时落地 | 重测 |

### 5.2 事后质量审核

**数据采集完成后进行**：

#### 5.2.1 视频质量审核清单

| 编号 | 审核项目 | 评分标准 |
|------|---------|---------|
| V1 | 分辨率达标 | 1080p = 合格，720p = 可接受 |
| V2 | 帧率达标 | 60fps = 优秀，30fps = 合格 |
| V3 | 画面稳定 | 无抖动 = 合格 |
| V4 | 曝光正常 | 细节清晰可见 = 合格 |
| V5 | 遮挡情况 | 无遮挡 = 合格 |
| V6 | 时间同步 | 双视角同步 = 合格 |

#### 5.2.2 动作质量审核清单

| 编号 | 审核项目 | 评分标准 |
|------|---------|---------|
| M1 | 起跳方式 | 自然跳下 = 合格 |
| M2 | 落地方式 | 双脚同时 = 合格 |
| M3 | 反跳执行 | 有明显反跳 = 合格 |
| M4 | 动作连贯 | 无明显停顿 = 合格 |
| M5 | 努力程度 | 明显用力 = 合格 |

### 5.3 剔除标准

以下情况的数据应被剔除：

1. **技术问题**：
   - 视频文件损坏或无法打开
   - 严重的画面模糊或曝光问题
   - 双视角不同步（时间差>50ms）
   - 受试者身体部分出画面

2. **动作问题**：
   - 单脚先落地
   - 落地后无反跳或反跳延迟明显
   - 动作中断或失败
   - 明显的非自然动作（如刻意表演）

3. **其他问题**：
   - 受试者穿着不符合要求
   - 有明显遮挡（他人、器材）
   - 受试者报告测试时身体不适

### 7.2 标注流程

#### 7.2.1 标注人员要求

- 运动医学、康复医学、运动科学背景
- 熟悉LESS评分标准
- 经过统一培训和一致性测试

#### 7.2.2 标注培训

1. LESS评分标准讲解（2小时）
2. 标准视频示例讨论（1小时）
3. 试标注练习（20例）
4. 一致性测试（通过率>80%方可参与正式标注）

#### 7.2.3 双人标注与仲裁

```
标注流程：

视频 → 标注员A独立标注 → 结果A
     → 标注员B独立标注 → 结果B

如果 结果A == 结果B：
    采用该结果
如果 结果A != 结果B：
    专家仲裁 → 最终结果
```

#### 7.2.4 标注工具

推荐使用专业视频标注工具：
- ELAN（语言学标注工具，可用于时间标注）
- BORIS（行为观察标注）
- 自定义Excel/Web标注界面

### 7.3 标注质量控制

| 指标 | 计算方法 | 目标值 |
|------|---------|--------|
| 评分者间信度 | Cohen's Kappa / ICC | >0.75 |
| 评分者内信度 | 重复标注一致性 | >0.85 |
| 关键帧误差 | IC/MKF帧差异 | <3帧(60fps) |
