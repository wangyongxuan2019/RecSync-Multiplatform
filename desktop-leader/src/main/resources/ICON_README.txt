图标文件说明
=============

要为RecSync Leader设置自定义图标，请将图标文件放置在此目录中：

Windows (.ico):
  文件名: icon.ico
  推荐尺寸: 256x256 像素
  格式: ICO格式

macOS (.icns):
  文件名: icon.icns
  推荐尺寸: 512x512 像素
  格式: ICNS格式

Linux (.png):
  文件名: icon.png
  推荐尺寸: 512x512 像素
  格式: PNG格式

如果没有提供图标文件，打包程序将使用默认的Java图标。

在线图标转换工具推荐:
- https://www.icoconverter.com/ (转换为ICO)
- https://cloudconvert.com/ (多格式转换)

建议：
- Leader和Client可以使用不同颜色的图标以便区分
- Leader建议使用蓝色或绿色图标（控制端）
- Client建议使用橙色或红色图标（录制端）
